## Installation
Add this line to your application's Gemfile:

```ruby
gem 'activerecord-destroyable', '~> 0.1.0'
```

And then execute:
```bash
$ bundle update
```

Or install it yourself as:
```bash
$ gem install activerecord-destroyable
```
